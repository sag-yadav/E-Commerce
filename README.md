# 🛒 E-Commerce Web Application

A modern and secure **E-Commerce Web Application** developed using **Spring Boot**, **Spring Security**, **Spring MVC**, **Spring Data JPA**, **Hibernate**, and **Maven**. The application provides essential e-commerce features such as user authentication, product management, shopping cart, and order processing.

The frontend is built using **Thymeleaf**, **HTML**, **CSS**, and **JavaScript** to deliver a responsive and user-friendly interface. The project follows the **MVC architecture** and uses **JPA & Hibernate** for efficient database management.

## 🚀 Tech Stack

* Spring Boot
* Spring Security
* Spring MVC
* Spring Data JPA
* Hibernate
* Maven
* Thymeleaf
* HTML, CSS, JavaScript
* MySQL

## ✨ Features

* User Authentication & Authorization
* Product Management
* Shopping Cart System
* Order Management
* Responsive UI
* Secure Database Integration
